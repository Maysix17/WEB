#  API NestJS Agro TIC

Gestion de Agrotic

### Instalación y Configuración

1. **Navegar al directorio del proyecto**
Abre una terminal y ejecuta:
   ```bash
   cd web
   cd MovilAgrotic
   cd ProyectoFormativo
   cd FrontEnd_AgroTic
   ```

2. **Instalar dependencias**
   ```bash
   npm install
   ```
2. **Ejecutar FrontEnd_AgroTic**
   ```bash
   npm run dev
   ```
se usa para gestionar los permisos para la apk
