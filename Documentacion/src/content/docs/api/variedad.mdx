- **URL:** http://localhost:3000/variedades

  - Content-Type: application/json
- **Body:** (raw JSON)

**Cuerpo de la solicitud:**
```json
{
  "nombre": "Cherry",
  "descripcion": "Tomate cherry pequeño",
  "cultivoId": "uuid-del-cultivo"
}

    "descripcion": "Tomate cherry pequeño",
    "cultivoId": "uuid-del-cultivo"
  }
}
```
