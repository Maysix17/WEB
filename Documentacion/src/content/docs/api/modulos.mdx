Crea un nuevo módulo.

**Requiere autenticación y autorización**

**Ejemplo en Postman:**
- **Método:** POST
- **URL:** http://localhost:3000/modulos
- **Headers:**
  - Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4gRG9lIiwiaWF0IjoxNTE2MjM5MDIyfQ.SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c
  - Content-Type: application/json
- **Body:** (raw JSON)

**Cuerpo de la solicitud:**
```json
{
  "nombre": "Reportes",
  "descripcion": "Módulo para generación de reportes del sistema"
}
```

**Respuesta exitosa (201):**
```json
{
  "success": true, // Indica que la operación fue exitosa
  "data": {
    "id": "uuid", // Identificador único del módulo creado (primary key, UUID)
    "nombre": "Reportes", // Nombre del módulo del sistema
    "descripcion": "Módulo para generación de reportes del sistema" // Descripción detallada de las funcionalidades del módulo
  }
}
```
