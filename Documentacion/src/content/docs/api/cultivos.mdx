# Cultivos

## POST /cultivos

Crea un nuevo cultivo.

**Requiere autenticación**

**Ejemplo en Postman:**
- **Método:** POST
- **URL:** http://localhost:3000/cultivos
- **Headers:**
  - Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4gRG9lIiwiaWF0IjoxNTE2MjM5MDIyfQ.SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c
  - Content-Type: application/json
- **Body:** (raw JSON)

**Cuerpo de la solicitud:**
```json
{
  "nombre": "Tomate", // Nombre del cultivo a registrar
  "descripcion": "Cultivo de tomate cherry", // Descripción detallada del cultivo
  "tipoCultivoId": "uuid-del-tipo", // ID del tipo de cultivo (perenne/transitorio)
  "zonaId": "uuid-de-la-zona" // ID de la zona donde se ubicará el cultivo
}
```

**Respuesta exitosa (201):**
```json
{
  "success": true,
  "data": {
    "id": "uuid",
    "nombre": "Tomate",
    "descripcion": "Cultivo de tomate cherry",
    "tipoCultivoId": "uuid-del-tipo",
    "zonaId": "uuid-de-la-zona"
  }
}
```
